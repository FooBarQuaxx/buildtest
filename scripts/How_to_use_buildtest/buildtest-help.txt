-bash-4.2$ _buildtest --help
usage: buildtest [options]

optional arguments:
  -h, --help            show this help message and exit
  -V, --version         show program version number and exit
  --logdir LOGDIR       Path to write buildtest logs. Override configuration BUILDTEST_LOGDIR
  --testdir TESTDIR     Path to write buildtest tests. Overrides configuration BUILDTEST_TESTDIR
  --ignore-easybuild    ignore if application is not built with easybuild
  --show                show buildtest environment configuration
  --clean-build         delete software test directory before writing test scripts
  --show-keys           display yaml key description

Basic Options:
  buildtest basic options

  -mns {HMNS,FNS}, --module-naming-scheme {HMNS,FNS}
                        Specify module naming scheme for easybuild apps
  --scantest            Report all tests that can be built with buildtest by checking all available apps found in eb
                        stack and system packages
  --clean-logs          delete buildtest log directory ($BUILDTEST_LOGDIR)
  --clean-tests         delete testing directory ($BUILDTEST_TESTDIR)

Find Options:
  buildtest options for finding software, toolchains, tests, yaml files

  -lt, --list-toolchain
                        retrieve a list of easybuild toolchain used for --toolchain option
  -ls, --list-unique-software
                        retrieve all unique software found in your module tree specified by BUILDTEST_MODULE_ROOT
  -svr, --software-version-relation
                        retrieve a relationship between software and version found in module files
  -fc FINDCONFIG, --findconfig FINDCONFIG
                        Find buildtest YAML config files found in BUILDTEST_CONFIGS_REPO. To find all yaml config
                        files use -fc all
  -ft FINDTEST, --findtest FINDTEST
                        Find test scripts generated by buildtest defined by BUILDTEST_TESTDIR. To find all test
                        scripts use -ft all
  -ecmt, --easyconfigs-in-moduletrees
                        Return a list of easyconfigs from a easybuild module tree
  --diff-trees DIFF_TREES
                        Show difference between two module trees

Test Options:
  Options for building tests with buildtest

  -s INSTALLED-EASYBUILD-APPS, --software INSTALLED-EASYBUILD-APPS
                        Specify software package to test
  -t INSTALLED-EASYBUILD-TOOLCHAINS, --toolchain INSTALLED-EASYBUILD-TOOLCHAINS
                        Specify toolchain for the software package
  --shell {sh,bash,csh}
                        Select the type of shell when running test
  --system SYSTEM-PACKAGE
                        Build test for system packages To build all system package test use --system all
  --module-load-test    conduct module load test for all modules defined in BUILDTEST_MODULE_ROOT
  --python-package PYTHON-PACKAGES
                        build test for Python packages
  --r-package R-PACKAGES
                        build test for R packages
  --ruby-package RUBY-PACKAGES
                        build test for Ruby packages
  --perl-package PERL-PACKAGES
                        build test for Perl packages

YAML Options:
  Options for YAML configuration

  --sysyaml INSTALLED-SYSTEM-PACKAGE
                        generate YAML configuration for binary test for system package
  --ebyaml YAML-APP-CHOICES
                        generate YAML configuration for binary test for software package

Job Scheduler Options:
  Options for interacting with Job Scheduler

  --enable-job          enable job script generation with buildtest
  --job-template JOB_TEMPLATE
                        specify job template file to create job submission script for the test to run with resource
                        scheduler
  --submitjob SUBMITJOB
                        specify a directory or job script to submit to resource scheduler

Miscellaneous Options:
  rest of buildtest options

  --runtest             Run the test interactively through runtest.py
